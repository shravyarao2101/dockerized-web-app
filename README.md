\# Dockerized Web Application



This project demonstrates containerization of a simple web application using Docker.



\## Goals

\- Learn Docker fundamentals

\- Build custom Docker images

\- Run applications in containers

\- Prepare for CI/CD and Kubernetes workflows



\## Planned Features

\- Custom Dockerfile

\- Exposed ports

\- Docker Compose setup

\- Production-style README



This project is part of my structured DevOps learning roadmap.
